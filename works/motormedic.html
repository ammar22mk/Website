MotorMedic
"A Doctor For Your Ride"

A comprehensive Flutter mobile app designed to educate and empower users in motorcycle maintenance

Project Overview

MotorMedic was developed as a comprehensive tool designed to educate and empower users in motorcycle maintenance. This mobile application provides motorcycle owners with the knowledge and tools they need to properly maintain their vehicles, diagnose common issues, and understand their bike better.

This was a group project for our Software Engineering subject, where we applied software engineering principles and methodologies to create a practical, user-friendly mobile application.

Technologies Used

Flutter Dart Mobile Development Cross-platform

Project Context

Course: Software Engineering

Type: Group Project

This project allowed us to apply theoretical knowledge from our Software Engineering course into practical application. We followed proper software development lifecycle, from requirements gathering to design, implementation, testing, and deployment.

Key Features

  • Comprehensive motorcycle maintenance guides and tutorials
  • Educational content for different types of motorcycles
  • User-friendly interface designed for motorcycle enthusiasts
  • Cross-platform mobile application (Android & iOS)
  • Practical maintenance tips and best practices

What I Learned

  • Mobile application development using Flutter framework
  • Dart programming language fundamentals and best practices
  • Cross-platform mobile development techniques
  • Team collaboration and version control in group projects
  • Software engineering principles and methodologies
  • UI/UX design for mobile applications
